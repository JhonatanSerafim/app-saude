# admin consulta fácil

> Sistema de gerenciamento do APP Consulta Fácil

## Build Setup

```bash
# Clonar projeto
git clone 

# Entre no diretório
cd 

# Instale as dependencias 
npm install

# Inciando projeto
npm run dev
```

Abrirá automaticamente nesta porta http://localhost:3000
